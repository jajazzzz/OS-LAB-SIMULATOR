ABOUT

In an Operating System that uses paging for memory management, a page replacement algorithm is needed to decide which page needs to be replaced when a new page comes in.

Page Fault - occurs when a program tries to access a memory page that is not present in the memory.
Page Hit - occurs when a program tries to access a memory page that is already present in the memory.

ALGORITHMS

FIFO

Oldest page is replaced by the new page.
LRU

Least recently used page is replaced by the new page.
